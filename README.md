mpush
========================

Push Notification Server written by Go.

## Prerequisite

- install Go
- setup GOPATH and GOROOT
- prepare cert.pem, and key.pem

## Usage

    $ ./setup.sh
    $ go build -o mpush
    $ ./mpush # listening to 8080 port.

## Contribution
